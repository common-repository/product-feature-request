Product Feature Request plugin allows you to easily create and manage feature requests in your WordPress products.

== Description ==

Many a time, it’s difficult to incorporate all of the features in a WooCommerce Product. Chances are that the missed out feature could be a significant one that would bring in more conversion rates for your online store! Product Feature Request - the WordPress idea management plugin is a perfect solution for all your product feedbacking concerns.

With this brand new feature request plugin, you can easily collect ideas and suggestions from your customers so as to include new impressive features or improvements for the WooCommerce products.

== Installation ==

= Minimum Requirements =
* WooCommerce 5 or greater
* WordPress 5.2 or greater
* PHP 7.4 or greater
* MySQL 5.7 or greater

= Automatic installation =
1. Log in to your WordPress dashboard.
2. Navigate to the Plugins menu, and click "Add New".
3. Search and locate 'Product Feature Request' plugin.
4. Click 'Install Now', and WordPress will take it from there.

= Manual installation =
The manual installation method requires downloading the 'Product Feature Request' plugin and uploading it to your webserver via your FTP application. The WordPress codex contains [instructions on how to do this here](https://wordpress.org/support/article/managing-plugins/#manual-plugin-installation).

== Frequently Asked Questions ==

= How can I create and manage Feature Requests? =
Select any Product on the frontend → Feature Request tab -> post Request and will create a new Feature request for the product. To manage, you can All Feature Request → Edit the Feature request.

= Product Feature Request Plugin" for WordPress responsive? =
Product Feature Request Plugin is completely responsive

= How many requests can I add in one product? =
You can add as many requests as you need. There is no limit defined.
